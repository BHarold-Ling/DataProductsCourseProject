Instructions


 


In
the control panel in the upper left, there are three controls.


    
	
  • You
	can select any country that is in the database; this is limited to
	countries and regions that have report COVID-19 cases.
		
    
	
  • You
	can select the date that you want reported.  Any cases later
	than the selected date will not be included in the charts. 
	This defaults to the latest date in the database. 
	
    
	
  • You
	can select the number of days to be considered recent.
	If you select 1, then only the report date will be counted as recent
	in the charts.  If you select 7 days (the default), then the
	week ending on the report date will be considered recent.
		
    



 


Notes:


    
	
  • This
	data comes from the European Centre for Disease Prevention and
	Control,
	https://www.ecdc.europa.eu/en/publications-data/download-todays-data-geographic-distribution-covid-19-cases-worldwide.
		
    
	
  • The
	dates and numbers from this source do not exactly match other
	sources.  It takes time for the reports from each original
	administrations to be disseminated, and there are differences in the
	way different sources count cases.
